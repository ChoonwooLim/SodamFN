## 1. 제품 개요

### 1.1 SodamFN이란?

SodamFN(소담 파이낸스)은 **소규모 외식업 사업자**를 위한 **올인원 클라우드 경영관리 플랫폼**입니다. 매출·매입·손익·인사·급여·계약을 하나의 시스템에서 통합 관리하며, **AI 자동 분류**, **GPS 기반 출퇴근**, **카카오 알림톡 연동**, **전자계약 서명** 등 프리미엄 기능을 제공합니다.

### 1.2 기술 스택

| 구분 | 기술 |
|------|------|
| **프론트엔드** | React 19 + Vite, Recharts (차트), Lucide Icons, Framer Motion |
| **백엔드** | Python FastAPI, SQLModel ORM |
| **데이터베이스** | PostgreSQL (클라우드/자체 서버), SQLite (로컬 폴백) |
| **인증** | JWT 토큰 + Google/Naver/Kakao 소셜 로그인 |
| **알림** | Solapi 카카오 알림톡 API |
| **지도** | Google Maps API (Geofence) |
| **배포** | Render.com / 자체 Orbitron 서버 |
| **앱 형태** | PWA (Progressive Web App) — 설치 없이 모바일 사용 |

### 1.3 사용자 역할

| 역할 | 설명 | 접근 가능 메뉴 |
|------|------|----------------|
| **관리자 (Admin)** | 사업주/매니저 | 대시보드, 매출관리, 매입관리, 카드매출분석, 손익계산서, 직원관리, 레시피북, 거래처관리, 설정 |
| **직원 (Staff)** | 알바/정규직 | 직원 대시보드(출퇴근), 내 전자계약, 설정 |

## 3. 핵심 기능 상세

### 3.1 📊 대시보드 (`/dashboard`)

관리자가 로그인하면 가장 먼저 보는 **경영 현황 종합 화면**입니다.

#### 기능 목록

- **KPI 카드 4종**: 월 매출, 월 지출, 순이익, 이익률(%) — 전월 대비 증감 표시
- **매출 추이 차트**: Area Chart로 최근 12개월 매출·지출·이익 트렌드 시각화
- **매출 구성 파이 차트**: 매장매출/배달앱매출 비율 시각화 (Recharts PieChart)
- **월 선택**: 연도·월 탐색으로 과거 데이터 조회
- **실시간 데이터**: API에서 월별 요약, 매출 분석, Top 지출처 데이터를 실시간 로드
- **포맷팅**: 금액은 한국 원화(₩) 포맷, 천 단위 콤마 자동 적용

### 3.2 💰 매출 관리 (`/revenue`)

**매장매출 + 배달앱매출(쿠팡이츠, 배달의민족, 요기요, 땡겨요)을 통합 관리**합니다.

#### 기능 목록

- **일별 매출 기록**: 날짜별 매출 등록/수정/삭제 (CRUD)
- **일별 그리드 뷰**: 31일 × 거래처/채널 매트릭스 형태의 Excel형 인라인 편집 그리드
- **매장매출 구분**: 다점포 지원 — 매장별(본점, 2호점 등) 매출 분리 관리
- **배달앱 정산**: 쿠팡이츠/배민/요기요/땡겨요별 정산금, 수수료, 주문건수 추적
- **Excel 업로드**: 배달앱 정산서 Excel 파일을 업로드하면 자동 파싱하여 매출 등록
- **P/L 연동**: 매출 데이터가 손익계산서의 수입 항목에 자동 반영
- **카드별 색상 구분**: 카드사(신한, KB, 하나 등)별 고유 색상 뱃지
- **월 탐색**: 이전/다음 달 네비게이션
- **접기/펼치기**: 일자별 카드 접기로 대량 데이터도 깔끔하게 정리

### 3.3 🛒 매입 관리 (`/purchase`)

**카드 결제 내역(매입 명세서)을 Excel로 업로드하고, AI 자동 분류로 비용 카테고리를 배정**합니다.

#### 기능 목록

- **Excel 업로드**: 카드사 매입 명세서 Excel/CSV 파일 파싱 (financial_parser, purchase_parser 서비스)
- **AI 자동 분류 (SmartClassifier)**:
  - 사용자가 가맹점의 카테고리를 수정하면 **규칙을 자동 학습**
  - 다음 업로드 시 동일 가맹점에 **학습된 카테고리를 자동 적용**
  - 정확 매칭 + 부분 매칭 이중 전략
  - 신뢰도(confidence) 점수 기반 우선순위
- **12개 비용 카테고리**: 원재료비, 소모품비, 수도광열비, 임차료, 수선비, 감가상각비, 세금과공과, 보험료, 인건비, 카드수수료, 기타경비, 개인가계부
- **카테고리 도우미**: 각 카테고리마다 해당하는 항목 가이드 제공 (한국 개인사업자 회계기준)
- **사업/개인 전환**: 한 번의 토글로 사업비 ↔ 개인가계부 전환 (P/L 제외)
- **일괄 처리**: 체크박스로 다중 선택 → 일괄 카테고리 변경, 일괄 삭제
- **업로드 이력 관리**: 업로드별 건수·날짜 추적, **롤백(되돌리기)** 기능으로 잘못된 업로드 일괄 취소
- **CRUD**: 개별 거래 수동 추가/수정/삭제
- **카드사 색상 뱃지**: 신한(파랑), KB(노랑), 현대(회색) 등 시각 구분

### 3.4 💳 카드 매출 분석 (`/finance/card-sales`)

**카드사별 승인 내역과 입금 내역을 분석**합니다.

#### 기능 목록

- **승인 내역 업로드**: 카드 승인 Excel 파일 파싱 — 승인일자, 카드사, 카드번호, 승인번호, 금액, 할부, 상태(승인/취소)
- **입금 내역 업로드**: 카드 입금 Excel 파일 파싱 — 입금일자, 카드사, 매출금액, 수수료, 부가세, 실입금액, 입금은행
- **카드사별 통계**: 막대 차트(BarChart)로 카드사별 승인 금액 비교
- **매출 구성 파이 차트**: 카드사별 매출 비율 시각화
- **월별 필터링**: 연도·월 선택으로 기간별 분석
- **합계 KPI**: 총 승인금액, 총 입금액, 총 수수료 요약 카드

### 3.5 📈 손익계산서 (`/finance/profitloss`)

**12개월 손익현황을 Excel형 테이블로 조회·편집하는 핵심 재무 모듈**입니다.

#### 기능 목록

- **대시보드 뷰**: 월별 매출·지출·이익 트렌드 차트, 이익률 추이, KPI 요약카드
- **12개월 P/L 테이블**: 1~12월의 모든 수입·지출 항목을 한눈에 보는 스프레드시트
  - **수입 항목**: 매장매출, 쿠팡 정산금, 배민 정산금, 요기요 정산금, 땡겨요 정산금
  - **지출 항목**: 인건비(자동), 퇴직금적립(인건비×10% 자동), 원재료비, 소모품비, 수도광열비, 임차료, 수선비, 감가상각비, 세금과공과, 보험료, 카드수수료, 기타경비
  - **자동 집계**: 합계, 평균 자동 계산
- **인라인 편집**: 셀 클릭 → 직접 금액 입력 → 저장 (Excel처럼 사용)
- **세부 지출 뷰**: 월별 거래처×일자 매트릭스 (30일 × N 거래처) 그리드
  - 카테고리별 소계, 일자별 합계 자동 집계
  - 셀 단위 인라인 편집
- **월별 분석**: 수입/지출 구성비 차트, 전월 대비 분석
- **거래처 연동**: VendorSettings에서 설정한 거래처 목록이 자동으로 행(row)으로 표시
- **인건비 자동 동기화**: Payroll 모듈에서 산출된 급여 합계가 P/L 인건비에 자동 반영

### 3.6 👥 직원 관리 (`/staff`, `/staff/:id`)

**직원 인사정보, 근태, 급여, 전자계약, 서류를 통합 관리**합니다.

#### 3.6.1 직원 목록 (`/staff`)

- **직원 카드 리스트**: 이름, 역할, 상태(재직/퇴사), 계약유형 표시
- **검색/필터**: 이름 검색, 상태 필터(재직/퇴사/전체)
- **신규 직원 등록 모달**: 성명, 직책, 시급, 입사일, 이메일, 급여계좌 입력

#### 3.6.2 직원 상세 (`/staff/:id`)

거대한 통합 인사카드로 다음 모든 정보를 한 화면에서 관리:

**기본 정보 탭:**

- 성명, 연락처(자동 포맷팅), 이메일, 국적, 비자종류
- 주소, 주민등록번호
- 계약유형(아르바이트/정규직/일용직), 입사일, 상태
- 4대보험 가입여부, 보수월액
- 시급, 월급, 부양가족 수, 자녀 수(8~20세)

**급여 계좌:**

- 은행명, 계좌번호, 예금주 별도 관리

**근로계약 상세:**

- 계약기간(시작~종료일)
- 근무시간(시작~종료, 휴게시간)
- 근무일/주휴일
- 직무내용
- 상여금(유무, 금액)
- 급여 지급일, 지급방법

**서류 관리:**

- 근로계약서, 보건증, 신분증 사본, 통장 사본 업로드
- 체크리스트로 제출 현황 관리
- 파일 다운로드/미리보기

**전자계약:**

- 근로계약서 자동 생성 (직원 정보 템플릿 변수 자동 치환)
- 카카오 알림톡으로 서명 요청 발송
- 서명 상태 추적 (대기/서명완료)

**급여 내역:**

- 월별 급여 산출 이력 목록
- 급여명세서 보기 (A4 규격 PayrollStatement 모달)
- 급여이체 상태 추적 (대기/완료/실패)

**근태 입력 (AttendanceInput):**

- 월별 달력 그리드로 일자별 근무시간 입력
- 일별 상태: 출근, 결근, 유급휴일, 무급휴일, 공휴일
- 임시공휴일 지정 기능
- 급여 산출 버튼 → 산출 완료 시 급여명세서 자동 표시

**계정 관리:**

- 직원에게 로그인 계정 생성 (아이디/비밀번호/등급)
- 등급: normal, vip, vvip, admin

### 3.7 💵 급여 산출 엔진 (Payroll Calculation)

**2026년 한국 세법 기준의 정밀 급여 자동 산출 시스템**입니다.

#### 산출 항목

- **기본급**: 시급 × 근무시간 (아르바이트) 또는 월급 (정규직)
- **주휴수당**: 주별(W1~W5) 주휴수당 자동 계산 — 주 15시간 이상 근무 시 발생
- **식비 지원**: 비과세 식비 자동 반영
- **4대보험 공제**: 보수월액 기준
  - 국민연금(NP): 4.5%
  - 건강보험(HI): 3.545%
  - 장기요양보험(LTI): 건강보험의 12.95%
  - 고용보험(EI): 0.9%
- **소득세 공제**: 부양가족 수, 자녀 수 기반 간이세액표 적용
- **지방소득세**: 소득세의 10%
- **총 지급액**: 기본급 + 수당 - 공제액

### 3.8 📄 급여 명세서 (PayrollStatement)

**A4 규격의 전문적인 급여지급명세서를 화면에 표시하고 인쇄**할 수 있습니다.

#### 기능 목록

- **A4 규격 레이아웃**: 실제 인쇄 시 A4 용지에 정확히 맞는 규격
- **지급 항목**: 기본급, 주휴수당(W1~W5 상세), 식비지원
- **공제 항목**: 국민연금, 건강보험, 장기요양, 고용보험, 소득세, 지방소득세
- **6:4 비율**: 지급항목(좌측 60%)과 공제항목(우측 40%) 가독성 최적화
- **반응형 스케일링**: useRef + useState로 화면 크기에 자동 맞춤
- **인쇄 버튼**: 브라우저 인쇄 기능 연동
- **근무 내역 표시**: 일자별 근무시간 상세 기록

### 3.9 📱 직원 대시보드 (`/staff-dashboard`) — PWA 앱

**직원이 스마트폰으로 사용하는 전용 앱**입니다.

#### 기능 목록

- **GPS 출퇴근**: 스마트폰 GPS 위치 기반 출근/퇴근 기록
- **Geofence 검증**: 매장 반경(기본 100m) 이내에서만 출퇴근 인정
  - 위치 검증 성공/실패 상태 표시
  - 매장과의 거리(m) 표시
- **출퇴근 상태**: 미출근 → 출근완료 → 퇴근완료 상태 추적
- **오늘의 근무 요약**: 출근시간, 퇴근시간, 총 근무시간
- **최근 근태 이력**: 최근 5건의 출퇴근 기록
- **급여 요약**: 이번 달 예상 급여, 근무일수, 시급 정보
- **내 전자계약**: 서명 대기 중인 계약서 목록
- **카메라 QR 스캔**: QR코드 기반 출퇴근 (추가 확장 기능)
- **PWA 설치**: 홈 화면에 추가하여 네이티브 앱처럼 사용

### 3.10 ✍️ 전자계약 서명 (`/contracts/sign/:id`)

**근로계약서를 디지털로 서명하는 전자계약 시스템**입니다.

#### 기능 목록

- **계약서 표시**: 표준근로계약서 전문 표시 (직원 정보 자동 치환)
- **서명 방식 2가지**:
  - **직필 서명**: Canvas 기반 터치/마우스 서명 — 펜 두께, 색상 조절
  - **자동 직인 생성**: 이름 기반 빨간 원형 직인 자동 생성 (Canvas drawText)
- **서명 지우기/재작성**: 서명 초기화 기능
- **서명 제출**: 서명 데이터(Base64) + 계약서를 서버에 저장
- **카카오 알림톡 연동**: 관리자가 서명 요청을 카카오톡으로 발송

### 3.11 📖 소담 레시피 (`/recipes`)

**매장 메뉴별 조리 레시피를 디지털로 관리**합니다.

- **레시피 카드**: 메뉴명, 이모지 아이콘, 카테고리 표시
- **상세 레시피**: 재료(정확한 그램 단위), 조리 순서(단계별)
- **이미지 라이트박스**: 레시피 이미지 확대 보기, 좌우 네비게이션
- **실제 레시피**: 어묵조림, 참치마요, 매콤참치, 불고기, 스팸주먹밥 등 실제 매장 레시피

### 3.12 🏢 거래처 관리 (`/vendor-settings`)

**매입처(비용)와 매출처(수입) 거래처를 통합 관리**합니다.

#### 기능 목록

- **매입처/매출처 분리**: 탭으로 비용 거래처 / 매출 거래처 구분 관리
- **카테고리별 그룹핑**: 원재료비, 소모품비 등 12개 비용 카테고리 + 배달앱/매장 매출 카테고리
- **CRUD**: 거래처 추가, 이름 수정(인라인 편집), 삭제
- **거래처 순서 변경**: 위/아래 이동으로 표시 순서 조정
- **거래처 병합(Merge)**: 동일 업체의 중복 거래처를 하나로 통합
  - 다중 선택 → 대표 거래처 지정 → 모든 거래 기록을 대표로 이관
- **미분류 거래처 관리**: Excel 업로드 시 자동 생성된 미분류 거래처 정리
- **다점포 지원**: 매장매출 거래처에 하위 매장(본점, 2호점) 추가
- **거래 통계**: 거래처별 최근 거래일, 거래건수, 총 거래금액 표시

  - Google Maps 지도에서 매장 위치 핀 지정
  - 현재 위치 사용 버튼
  - 허용 반경(m) 설정 (기본 100m)
  - 실시간 위도/경도 표시
- **거래처 관리**: VendorSettings 내장
- **계약서 템플릿 설정**: ContractSettings (표준근로계약서 템플릿 편집)

### 3.14 🔐 인증 시스템

#### 기능 목록

- **JWT 토큰 인증**: 로그인 시 JWT 발급, 모든 API 요청에 Bearer 토큰 첨부
- **역할 기반 접근 제어**: admin/staff 역할에 따라 메뉴 및 API 접근 제한
- **등급 시스템**: normal, vip, vvip, admin 4단계
- **소셜 로그인**: Google, Naver, Kakao OAuth 2.0 지원
- **자동 로그아웃**: 401 응답 시 세션 클리어 + 로그인 페이지 리다이렉트
- **프로필 이미지**: 소셜 로그인 시 프로필 사진 연동

### 3.15 📲 카카오 알림톡 (NotificationService)

#### 기능 목록

- **전자계약 서명 요청**: 직원에게 계약서 서명 링크를 카카오톡으로 발송
- **근태 입력 요청**: 월 마감 시 직원에게 근무시간 확인 요청 발송
- **급여명세서 발송**: 급여 산출 완료 후 명세서 링크를 카카오톡으로 발송
- **Solapi API v4 연동**: HMAC-SHA256 인증, 템플릿 기반 알림톡

### 3.16 🏦 급여이체 서비스 (BankingService)

#### 기능 목록

- **이체 검증**: 출금계좌(사업자), 입금계좌(직원) 정보 사전 검증
- **이체 실행**: 오픈뱅킹/펌뱅킹 API 연동 준비 완료 (인터페이스 구현)
- **일괄 이체 데이터**: 다수 직원의 급여를 Excel Export용 데이터로 일괄 준비
- **이체 상태 추적**: 대기 → 완료 → 실패 상태 관리

### 3.17 🤖 AI 자동 학습 분류 (SmartClassifier)

#### 동작 원리

1. **학습**: 사용자가 가맹점 카테고리를 수정하면 `VendorRule` 테이블에 규칙 저장
2. **적용**: 다음 Excel 업로드 시 학습된 규칙을 자동 적용
3. **강화**: 동일 규칙이 반복 확인되면 confidence 점수 증가
4. **매칭 전략**: 정확 매칭 우선 → 부분 매칭 폴백

#### 학습 트리거

- 카테고리 변경 (`category_change`)
- 거래처 병합 (`vendor_merge`)
- 사업/개인 전환 (`toggle_personal`)
- 거래처명 수정 (`vendor_patch`)

### 3.18 📊 데이터 입력 허브 (`/data-input`)

- **Excel 업로드**: 매출/매입 Excel 파일 드래그 앤 드롭
- **OCR 촬영**: 영수증 카메라 촬영 → OCR 텍스트 추출 → 자동 데이터 입력
- **업로드 이력**: 업로드별 파일명, 건수, 상태 관리 + 롤백 기능

  1. 카드 명세서 분류에 매월 3시간+ 소요
  2. 급여 계산 오류 → 직원 분쟁, 노동청 신고 리스크

## 부록: API 엔드포인트 전체 목록

### Auth (`/api/auth`)

- `POST /login` — 로그인 (JWT 발급)
- `POST /signup` — 회원가입
- `GET /me` — 현재 사용자 정보
- `GET /google/login` — Google OAuth
- `GET /naver/login` — Naver OAuth
- `GET /kakao/login` — Kakao OAuth

### HR (`/api/hr`)

- `GET /staff` — 전체 직원 목록 (검색/필터)
- `POST /staff` — 신규 직원 등록
- `GET /staff/{id}` — 직원 상세 (서류, 급여, 계약 포함)
- `PUT /staff/{id}` — 직원 정보 수정
- `DELETE /staff/{id}` — 직원 삭제 (관련 데이터 일괄 삭제)
- `POST /staff/{id}/account` — 직원 로그인 계정 생성
- `PUT /staff/{id}/account/grade` — 직원 등급 변경
- `POST /staff/{id}/document` — 서류 업로드
- `POST /attendance` — 출퇴근 기록 (GPS)
- `GET /attendance/status/{id}` — 오늘 출퇴근 상태
- `GET /attendance/history/{id}` — 최근 출퇴근 이력
- `GET /attendance/monthly-summary/{id}/{month}` — 월간 근무 요약
- `GET/POST /location` — 매장 위치 Geofence 조회/설정

### Payroll (`/api/payroll`)

- `POST /calculate` — 급여 산출
- `GET /list/{staff_id}` — 급여 이력
- `POST /transfer/{id}` — 급여 이체

### Finance (`/api/finance`)

- `GET /card-sales/stats` — 카드매출 통계
- `POST /card-sales/upload/approvals` — 승인내역 업로드
- `POST /card-sales/upload/payments` — 입금내역 업로드

### Revenue (`/api/revenue`)

- `GET/POST/PUT/DELETE /` — 매출 CRUD

### Purchase (`/api/purchase`)

- `GET/POST/PUT/DELETE /daily-expenses` — 매입 CRUD
- `POST /upload` — Excel 업로드
- `POST /rollback/{id}` — 업로드 롤백

### Profit/Loss (`/api/profitloss`)

- `GET /{year}` — 연간 손익 데이터
- `PUT /{year}/{month}` — 월별 P/L 수정

### Contract (`/api/contract`)

- `POST /create/{staff_id}` — 전자계약 생성
- `GET /{id}` — 계약서 조회
- `POST /{id}/sign` — 전자서명
- `POST /{id}/send-kakao` — 카카오톡 발송

### Settings (`/api/settings`)

- `GET/PUT /biz-account` — 사업자 계좌
- `GET/POST /rules` — AI 분류 규칙
