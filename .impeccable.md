## Design Context

### Users
- **주 사용자**: 40~50대 소규모 자영업자 (식당, 카페, 소매점 운영자)
- **IT 숙련도**: 낮음 - 스마트폰은 사용하지만 복잡한 소프트웨어에 익숙하지 않음
- **사용 맥락**: 바쁜 영업 중간중간, 또는 마감 후 매출/비용 확인. 한 손으로 조작하는 경우 많음
- **해결하는 문제**: 매출/매입 관리, 직원 급여, 손익 계산 등 사업 운영의 숫자 관리를 쉽게
- **직원 앱 사용자**: 20~30대 알바/직원. 출퇴근 기록, 급여 확인, 공지 열람

### Brand Personality
- **3단어**: 신뢰감, 심플, 따뜻함
- **톤**: 전문적이면서도 친근한. 은행 앱처럼 차갑지 않고, 게임 앱처럼 가볍지 않은 중간 톤
- **감정 목표**: "내 가게 관리가 이렇게 쉬울 수 있구나" - 안심감과 자신감

### Aesthetic Direction
- **Admin 대시보드**: 프로페셔널, 데이터 중심. 다크 네이비(#0F172A) 기반 + 블루(#3B82F6) 액센트
- **Staff 앱**: 모던, 친근, 모바일 최적화. 밝은 배경(#f0f4f8) + 틸/그린(#06D6A0) 액센트
- **공통**: Slate 팔레트 기반, 글래스모피즘 포인트, 부드러운 그림자, 라운드 코너
- **Anti-reference**: 복잡한 ERP 소프트웨어, 빽빽한 표와 작은 글씨, 전문 용어 가득한 UI
- **테마**: 라이트 모드 전용 (다크 모드 미지원)

### Design Principles

1. **큰 글씨, 넓은 터치 영역**: 40~50대 사용자가 안경 없이도 읽을 수 있는 크기. 최소 터치 타겟 48px. 중요한 숫자는 크고 굵게.

2. **한눈에 파악**: 복잡한 데이터도 시각적 위계를 명확히. KPI 카드 → 차트 → 상세 테이블 순서. 정보 과부하 방지.

3. **익숙한 패턴 사용**: 혁신적 UI보다 관습적 패턴 선호. 하단 탭 네비게이션, 카드 레이아웃, 풀다운 새로고침 등 스마트폰에서 익숙한 인터랙션.

4. **금액은 항상 명확하게**: 원화 포맷(쉼표), 만/억 축약(fmtShort), 증감 표시(초록/빨강). 금액 관련 UI는 오류 여지 최소화.

5. **부드럽지만 절제된 모션**: 페이지 전환과 카드 등장은 자연스럽게(fadeInUp). 과한 바운스/엘라스틱 금지. prefers-reduced-motion 존중.

### Typography
- **Admin**: Pretendard (한국어 최적화 산세리프)
- **Staff App**: Inter (깔끔한 모던 산세리프)
- **숫자/금액**: tabular-nums로 정렬, 굵은 웨이트(700-800)
- **본문**: 16px 기본, 줄간격 1.6 이상

### Color System
- **Primary**: Slate 900 (#0F172A) - 신뢰감, 전문성
- **Action**: Blue 500 (#3B82F6) - 주요 액션, CTA
- **Success/Positive**: Teal (#06D6A0) / Green (#10B981) - 수익, 긍정적 변화
- **Danger/Negative**: Red (#EF4444) - 지출 초과, 경고, 삭제
- **Warning**: Amber (#F59E0B) - 주의, 대기 상태
- **배경**: 연한 블루틴트 (#f0f4f8) - 순백보다 눈의 피로 감소

### Accessibility
- **목표**: WCAG 2.1 AA 수준
- **폰트 크기**: 최소 14px, 중요 정보 16px 이상
- **색상 대비**: 4.5:1 이상 (텍스트), 3:1 이상 (큰 텍스트/아이콘)
- **색상만으로 정보 전달 금지**: 아이콘, 텍스트 라벨 병행
- **모션 축소**: prefers-reduced-motion 미디어 쿼리 존중
